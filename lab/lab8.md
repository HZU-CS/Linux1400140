# Lab8 **linuxC编程基础（2）**

实验目的：

1. 掌握进程相关系统调用
2. 掌握线程相关系统调用
3. 掌握文件相关系统调用

实验要求：

1. 了解 linux 进程相关概念和明确进程的含义
2. 认识并了解并发执行的实质并掌握 fork()系统调用的相关编程
3. 了解 exec()，system()，wait()，waitpid()，kill()等相关系统调用的功能
4. 熟练文件系统调用的相关编程

实验内容：
